DateMe

Simple, non-invasive JavaScript date sanitisation.

The date is validated on blur, and converted into DD/MM/YYYY format from an array of accepted inputs.

Try it out now.

If the date is valid, the field is updated. If not, a class of error is applied to the input.

To implement, call the script on your page, and apply a class of dateMe to each input element you want validated. Simples 


<input type="text" class="dateMe" />
<script src="js/dateme.min.js"></script>

Accepted input formats:

  • DDMMYY
  • DDMMYYYY
  • D/M/YY
  • D/M/YYYY
  • DD/MM/YY
  • DD/MM/YYYY
  • D-M-YY
  • D-M-YYYY
  • DD-MM-YY
  • DD-MM-YYYY
  • D.M.YY
  • D.M.YYYY
  • DD.MM.YY
  • DD.MM.YYYY
